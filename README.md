## Featured Projects

### Multicore RISC-V CPU Design and Performance Analysis
Designed and evaluated single-cycle, pipelined, cached, and multicore RISC-V CPU architectures in SystemVerilog.  
Key results: 77.68 MHz multicore frequency, 140.55 us execution time, and 1.19x multicore speedup at LAT=6.

### AXI–Ramulator2 Memory Subsystem Integration
Integrated a SystemVerilog AXI wrapper with the C++ Ramulator2 DRAM simulator using DPI-C.  
Evaluated DDR4 and HBM3 configurations, burst transfers, backpressure, reorder buffering, and memory throughput.
